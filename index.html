Featured Projects

Smart Medication Reminder

The Medication Reminder project is a dedicated health assistant that uses a Real-Time Clock to manage a dose daily schedule. It features a robust alert system where a red LED and buzzer trigger at programmed times, requiring the user to confirm via a touch sensor or button. Settings like dose times and treatment progress are saved in EEPROM to ensure they persist through power outages.

Secure Home Smart Door

The SmartHome Security System focuses on perimeter protection by integrating multiple sensors to detect intruders. It uses a PIR sensor for motion and an ultrasonic sensor to measure a person's distance, only prompting for a password if someone is within range. If an incorrect code is entered three times on the keypad, the system triggers an "Intruder Alert" and sounds a high-decibel alarm.

Embedded C Secure

Automated Ventilation System

This device functions as an automated climate controller that monitors real-time temperature and humidity using temperature sensor. When the temperature exceeds 25°C, it activates an external cooling fan and a red warning LED. If the temperature falls below 24°C, it shuts the fan off and triggers a blue LED to indicate a low-temperature state.
